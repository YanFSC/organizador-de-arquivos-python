\# 📁 Organizador de Arquivos em Python



Script que organiza automaticamente arquivos de uma pasta em categorias como imagens, documentos, vídeos, entre outros.



\---



\## 🚀 Funcionalidades



\* Identifica arquivos pela extensão

\* Cria pastas automaticamente

\* Move arquivos para suas respectivas categorias



\---



\## 🛠️ Tecnologias



\* Python

\* os

\* shutil



\---



\## ▶️ Como executar



1\. Clone o repositório:

&#x20;  git clone https://github.com/YanFSC/organizador-de-arquivos-python



2\. Acesse a pasta:

&#x20;  cd organizador-de-arquivos-python



3\. Execute o script:

&#x20;  python main.py



\---

Antes:
